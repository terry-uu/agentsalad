How it works
Three steps. That's it.
1

Create an Agent

Pick any LLM provider — Claude, GPT, Gemini, Groq, or bring your own. Describe what it should do.

2

Connect a Channel

Paste a bot token for Telegram, Discord, or Slack. Your agent goes live on the messenger you already use.

3

Add a Target

A person for private DM, or a room for community-wide service. One agent can serve many targets.

What you can do
Your agent, everywhere.
👨‍👩‍👧

Family Agent

Make an agent for your parents. Private DM, simple and personal. They just talk to a bot — no setup on their end.

💬

Community AI

Drop an agent into your Discord server or Slack workspace. Everyone in the channel gets instant AI access.

📋

Team Workspace

Shared folders let team members collaborate through the agent. Collect tasks, share files, stay in sync — organically.

⏰

Scheduled Operations

Set up cron jobs for batch messaging, daily summaries, or recurring check-ins across all your targets at once.

Custom Skills

Extend your agent with any script or prompt.

Shell scripts — run.sh / run.cmd
JSON input/output — structured data flow

Custom input schema — typed parameters

Write a run.sh, define inputs, and your agent gets a new tool. Connect any API, query any database, automate anything — with zero framework code.

Philosophy
"Running your own AI agent should be as normal as running your own Wi-Fi router."

Not a luxury. Not a subscription. Just something you set up once and it works for you. One Node.js process, on a machine you already own. Your agent, your data, your rules.
